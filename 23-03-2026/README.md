# Day 1 - Control Flow and Loops (Java)

This folder contains beginner Java practice programs focused on:
- conditional statements (`if/else`, `switch`)
- loops (`for`, `while`, `do-while`)
- basic number/string logic
- pattern printing
- simple console interaction

## Folder Contents

- `ATMConsole.java`
- `FactorialWhile.java`
- `FibSeqGen.java`
- `GradeCategorizer.java`
- `LeapYear.java`
- `MultiplicationGen.java`
- `PallindromeCheck.java`
- `PasswordCheck.java`
- `PatternOne.java`
- `PatternTwo.java`
- `PrimeChecker.java`
- `VowelConsonant.java`

## Practice Questions Covered

1. **ATM Console Simulation**  
   File: `ATMConsole.java`  
   Features: check balance, withdraw, deposit, exit (menu-driven loop).

2. **Factorial Using While Loop**  
   File: `FactorialWhile.java`  
   Input: integer `n`  
   Output: factorial of `n`.

3. **Fibonacci Sequence Generator**  
   File: `FibSeqGen.java`  
   Input: number of terms  
   Output: first `n` Fibonacci numbers.

4. **Grade Categorizer**  
   File: `GradeCategorizer.java`  
   Input: marks  
   Output: grade (`A/B/C/D/F`) based on score range.

5. **Leap Year Checker**  
   File: `LeapYear.java`  
   Input: year  
   Output: whether the year is leap or not.

6. **Multiplication Table Generator**  
   File: `MultiplicationGen.java`  
   Input: number `n`  
   Output: multiplication table from `n x 1` to `n x 10`.

7. **Palindrome Number Check**  
   File: `PallindromeCheck.java`  
   Input: integer  
   Output: reversed number and palindrome status.

8. **Password Validator Using do-while**  
   File: `PasswordCheck.java`  
   Repeats input until correct password is entered.

9. **Pattern Printing - Left Triangle**  
   File: `PatternOne.java`  
   Input: number of rows  
   Output: increasing `*` triangle.

10. **Pattern Printing - Pyramid**  
    File: `PatternTwo.java`  
    Input: number of rows  
    Output: centered star pyramid.

11. **Prime Number Checker**  
    File: `PrimeChecker.java`  
    Input: positive integer  
    Output: whether number is prime.

12. **Vowel or Consonant Checker**  
    File: `VowelConsonant.java`  
    Input: character  
    Output: vowel/consonant/alphabet validation intent.
